************
Introducción
************
El modulo de clanes permite a los usuarios crear equipos donde pueden interactuar
entre ellos como equipo dentro del juego, tambien pueden realizar eventos para las
personas integrantes del grupo, o participar como equipo en actividades especificas
para grupos dentro del juego donde esté incluido dicho módulo.
